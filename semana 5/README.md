# Galería Interactiva con JavaScript

Este proyecto consiste en una galería de imágenes interactiva desarrollada con HTML, CSS y JavaScript. Permite agregar imágenes mediante una URL, seleccionar una imagen y eliminarla dinámicamente usando manipulación del DOM.

## Funcionalidades
- Agregar imágenes desde una URL
- Seleccionar una imagen con un clic
- Eliminar la imagen seleccionada
- Diseño responsive con CSS Grid
- Atajo de teclado (Delete) para eliminar imágenes

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (DOM y eventos)

## Publicación
Este proyecto puede ser visualizado mediante GitHub Pages.
